# E-Tailoring: Digital Bespoke Atelier

## Overview
E-Tailoring is a premium digital platform designed to bring the traditional bespoke tailoring experience into the modern era. Developed as a final year project, it focuses on high-precision digital measurements and artisan collaboration.

## Features
- **Digital Measurement System**: Seamlessly enter and track body measurements.
- **Master Tailor Selection**: Browse and connect with expert tailors.
- **Premium UI/UX**: A dark-themed, luxurious interface built with Next.js and Vanilla CSS.
- **Order Tracking**: Real-time updates on garment construction.

## Tech Stack
- **Frontend**: Next.js 15 (App Router), TypeScript
- **Styling**: Vanilla CSS (Premium Tokens)
- **Deployment**: Vercel (Recommended)
- **State**: React Hooks & Context
